Pre-registration

Preregistration Preregistration is the practice of documenting your research plan at the beginning of your study and storing that plan in a read-only public repository such as OSF Registries and As Predicted.org. Opensesame

OpenSesame is a program to create experiments for psychology, neuroscience, and experimental economics. It does not require coding skills to create psycholinguistics experiments and is particularly friendly to beginners with a comprehensive graphical, point-and-click interface. You can find tutorials and example experiments on the website that are highly useful in learning to build your own experiments. Opensesame experiments can be implemented both offline and online. It has an extension called OSWEB that allows the experiments to be excuted in a browser. ...
